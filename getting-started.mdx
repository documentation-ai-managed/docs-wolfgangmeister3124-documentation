## Basic Usage

You start your InstaDP experience by visiting the homepage and entering a public Instagram username. The platform retrieves profile details, including bio, posts, and images, without any authentication. This process takes less than `<5` seconds typically.

## Step-by-Step Instructions

Follow these steps to view and download content effectively.

<Steps>
  <Step title="Enter Username" icon="user">
    Navigate to instadp.com and input a username like `natgeo` in the search box.

    Press enter to initiate the fetch.
  </Step>
  <Step title="Browse Profile" icon="image">
    Examine the profile overview, including recent posts and highlights.

    Click on thumbnails to enlarge images.
  </Step>
  <Step title="Download Content" icon="download">
    Right-click images or use the download button below each post.

    For bulk actions, select multiple items and export as ZIP.
  </Step>
</Steps>

<Callout kind="info">
  Ensure the profile is public; InstaDP cannot access private accounts to maintain user privacy.
</Callout>

## Initial Setup

No installation required. Simply bookmark the site for quick access. If using the API, obtain a free key from the dashboard.

<ExpandableGroup>
  <Expandable title="What is a Public Profile?" default-open="true">

  </Expandable>
  <Expandable title="Download Limits">

## Best Practices

Avoid searching non-existent usernames to prevent errors. Use exact handles without `@` symbols. For large profiles with `>1000` posts, expect slightly longer load times.
