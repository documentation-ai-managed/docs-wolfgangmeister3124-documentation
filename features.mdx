## Integration and API

Extend functionality via the InstaDP API for developers.

<Expandable title="API Quickstart" default-open="false">
  Authenticate with your key and query profiles.

  ```javascript
  const apiKey = 'your_api_key';
  fetch(`https://api.instadp.com/profiles/${username}?key=${apiKey}`)
    .then(res => res.json())
    .then(data => {
      data.images.forEach(img => download(img.url));
    });
  ```
</Expandable>

## Privacy Features

InstaDP never stores searched profiles or downloads. Sessions expire after use, and all traffic uses HTTPS. You control data visibility completely.

| Feature | Benefit | Limit |
|---------|---------|-------|
| HD Downloads | Original quality | Free: 50/day |
| No Login | Anonymous access | Public only |
| Batch Export | Efficient saving | Up to 100 images |
